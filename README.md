|       选项       | BiShe  | JiSuanJiSheJiDaSai |
| :--------------: | :----: | :----------------: |
|      移动端      |  支持  |        支持        |
|       PC端       | 不支持 |        支持        |
|      主控端      |  支持  |        支持        |
| 危险行为识别数量 |   少   |         多         |

> **注**：移动端、PC端、主控端相互独立，可以随意组合。
